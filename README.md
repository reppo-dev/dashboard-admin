A dark-themed **admin dashboard UI** built primarily with **shadcn/ui** on top of **Next.js (App Router)** and **Tailwind CSS**.

> **UI-only project:** data is **mock/static** and there is **no authentication / backend** included.

---

## Highlights

- Dark dashboard layout with reusable UI components
- Sidebar navigation with grouped and nested items
- **Home/Dashboard** page with charts, cards, and widgets
- **User** page with profile info, badges, and activity chart

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui (Radix-based components)

---

## Getting Started

```bash
npm install
npm run dev
```
